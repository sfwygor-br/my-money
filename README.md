# my-money
App to control budget

### INSTALAÇÃO LOCAL
* Instale o Node JS;
* Instale o banco de dados MongoDB;
* Em seu servidor local baixe o projeto usando `git clone`;
* Instale as dependências através do `npm install`;

### BIBLIOTECAS DO PROJETO
* body-parser
* consign
* ejs
* espress
* express-session
* mongodb
